# Ultimate Quest
## Mod By IntelRyzen

Welcome to the beta version of the Ultimate Quest mod! This mod adds a few significant things to the game

*I am still learning NeoForge, so please report any bugs or glitches!*

## Requirements

- Minecraft Version: 1.21.1.
- Recommended: [NeoForge](https://neoforged.net/) 21.1.228 for Minecraft 1.21.1.
- Other NeoForge 1.21.1 versions may work, but 21.1.228 is the version this beta was built and tested against. Please report compatibility issues.

### Installation:

- Download `.jar` from Releases
- Put it in `%AppData%\.minecraft\mods` folder on Windows or `~/Library/Application Support/minecraft/mods` on macOS.
- Needs to be installed on both Server and Client

## Features

### Ultra Dust

**Ultra Dust** is a rare material found in the End dimension, typically generating near the lower Y-levels. 
1. Mine **Ultra Dust**.
2. Run it through a Processor (cannot be crafted in survival) to get **Processed Ultra Dust**.
3. Refine it with a Catalyst to forge **Ultra Ingots**.

You can craft the ingots into **Ultra Armor**, which with a full set allows you to enter the **Void Dimension**.


### Void Dimension

A barren wasteland of Bedrock, with almost no functionality right now - will be added soon.

You can teleport to it by falling in the void while wearing Ultra Armor.

### Bedrock Breaker

Crafted with **Obsidian-Processed Ultra Dust** in the standard pickaxe shape. 
* **Function:** Can break Bedrock by right-clicking.
* **Speed:** Slightly faster than a standard Netherite Pickaxe.

## Machines

***Note: The Processor and Refiner blocks currently cannot be crafted in survival mode. Their crafting recipes will be added in a future update!***

### Refiner

***The Refiner has a fluid input, and an item input***

Recipes:
- 1x *Processed Ultra Dust* w/ 250mB *Catalyst* => 1x *Ultra Ingot*

### Processor

***The Processor Block requires a redstone signal, and one ingredient.***

Recipes:
- 1x *Ultra Dust* => 1x *Processed Ultra Dust*
- 1x *Raw Gold* => 1x *Gold Ingot*
